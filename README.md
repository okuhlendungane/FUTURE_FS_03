# Gambu-Line Hiring Services Website
​
 This project is a website for Gambu-Line Hiring Services, offering mobile cold rooms, VIP & VVIP toilets, chairs, tables, and professional sound systems for various events.
​
 ## Project Structure
​
 The project is organized into the following main directories:
 - `html/`: Contains the main HTML pages (e.g., `index.html`, `services.html`).
 - `css/`: Contains the stylesheet (`style.css`) for the website's design.
 - `js/`: Contains JavaScript files (`script.js`) for interactive elements and custom components.
 - `img/`: Stores all images used on the website.
 - `php/`: Contains PHP scripts, specifically `form.php` for handling contact form submissions.
​
 ## Features
​ - **Responsive Design**: Adapts to various screen sizes (desktop, tablet, mobile).
 - **Sticky Navigation**: The navigation bar remains at the top of the viewport during scrolling.
 - **Animated Content**: Landing page elements ease into view with animations.
 - **Service Cards**: Displays services in an organized, visually appealing card format.
 - **About Section**: Dedicated section providing information about the business.
 - **Contact Form**: Allows users to send inquiries, with data submitted to a MySQL database.
## Technologies Used
​
 - HTML5
 - CSS3
 - JavaScript (ES6+)
 - PHP (for backend form handling)
 - MySQL (for database storage)
